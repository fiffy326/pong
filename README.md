# Pong

Clone of the 1972 Atari video game, written in C using SDL2.

## Controls

| Key       | Function          |
| --------- | ----------------- |
| W         | Left paddle up    |
| S         | Left paddle down  |
| UPARROW   | Right paddle up   |
| DOWNARROW | Right paddle down |

## Installation

```bash
git clone https://github.com/fiffy326/pong
cd primal
make
sudo make install
```

## Dependencies

- SDL2
